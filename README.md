# ai-motivational-quote-bot
AI-powered motivational quote email automation built with n8n and Groq AI.
## Features

- Automated execution using Schedule Trigger
- AI-generated motivational quotes
- Groq API integration
- Automated Gmail delivery
- No-code/Low-code workflow automation

## Workflow

Schedule Trigger → Groq AI → Data Processing → Gmail

## Tech Stack

- n8n
- Groq API
- Gmail API
- HTTP Requests
